# HCS-28 (Appendix): Adapter Catalog (Normative)

This appendix complements [HCS-28](../hcs-28.md) with per-adapter definitions.

Each adapter entry defines:

- adapter identifier;
- contribution mode;
- default weight;
- applicability;
- input contract; and
- score normalization to `[0,100]`.

Implementations MUST preserve adapter identifiers and score keys exactly when claiming HCS-28 baseline interoperability.

For the canonical adapter list, see [adapter index](./adapters/index.md).
